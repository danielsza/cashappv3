## 🔧 Application Descriptions

### CashServer.py
**Purpose:** Main server application  
**Features:**
- Controls COM port relay
- Authenticates users
- Processes client requests
- Logs all transactions
- Monitors peer servers
- UDP discovery listener

**Requirements:**
- Python 3.8+
- pyserial library
- Windows (or Linux)

### ServerConfig.py (NEW!)
**Purpose:** GUI configuration tool  
**Features:**
- 4-tab interface
- COM port detection
- Relay settings and testing
- Network configuration
- Logging setup
- Security settings

**Usage:**
- No more manual INI editing!
- Test COM port before deploying
- Test relay to verify hardware
- Easy for non-technical users

### UserManager.py
**Purpose:** User management utility  
**Features:**
- Add/remove users
- Change passwords
- Unlock accounts
- View user list
- Command-line interface

**Usage:**
- Create initial users
- Manage passwords
- Unlock locked accounts
- View all users

### CashClient.py
**Purpose:** Client GUI application  
**Features:**
- Windows/macOS/Linux compatible
- Auto-discover servers
- Transaction entry
- Canadian penny rounding
- BOD/EOD quick actions
- Settings management

**Usage:**
- Daily operations
- Opening cash drawers
- Entering transactions
- Managing configuration
